# MBMobileClient
Bem-vindo a versão APP mobile

## Telas Principais
- Tela de Login / Acesso
- Dashboard / Página inicial
- Listagem de registros
- Formulário de cadastro
- Formulário de edição
- Tela de detalhes
- Confirmação de exclusão

## Configurações: 
### [Expo](https://docs.expo.dev/ "Documentação") e [React Native](https://reactnative.dev/ "Documentação")

Para começar
1. Instale as dependências: 
`npm install`
2. Inicie o aplicativo: 
`npx expo start`
3. Para rodar o projeto: 
`npm run android`

- Tutoriais: [React Native em 2026](https://youtu.be/7uGjAMhI8G4 "guia prático e completo para começar do zero")
